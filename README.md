This repository consolidates the material for the CAS ADS. Each module has its own folder with lecture notes, presentations, exercises,datasets and README files.


### Modules Overview

- **Module 1 – Data Acquisition & Management**: Data sources, data types, data-management planning.  
- **Module 2 – Statistical Inference for Data Science**: Statistical methods for describing & analysing data; drawing scientific conclusions. 
- **Module 3 – Data Analysis & Machine Learning**: Practical application of ML methods using Python, on real data sets. 
- **Module 4 – Ethics & Best Practices**: Data ethics, reproducibility, good practices for code/data management, version control workflows (e.g. Git). 
- **Module 5 – Peer Consulting & Selected Readings**: Knowledge exchange, peer discussions, literature review and synthesis. 
- **Module 6 – Deep Learning**: Training and applying deep neural networks for a variety of tasks.
